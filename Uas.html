Languages C++

C++ is a computer programming language used to develop software, games, and system applications. C++ is an extension of the C language, but has some additional features, such as object-oriented programming (OOP). C++ was developed by Danish computer scientist Bjarne Stroustrup and released in 1985. Pros (Fast performance, portable, and manageable memory). Disadvantages (Development tools must be specific, language is too complex, and not suitable for beginners).
1. Numeric Data Types
Numeric data types in C++ are used to store numbers, both whole numbers and decimal numbers, including :

  • Int (integer) : for storing integers, such as -5, 0, or 12.
  • Float : for storing decimal numbers with low precision, such as 3.14 or -0.5.
  • Double : for storing decimal numbers with high precision, such as 3.14159265 or -0.12345678.
  • Long: used to store integers larger than int type.
  • Short: for storing integers smaller than int type.

2. Character Data Types
A character data type in C++ is a type used to store a character, consisting of one or more Unicode characters.
  • Char: used to store characters, such as 'a', 'b', or 'z'.
  • Wchar_t : used to store Unicode characters with a length of more than 8 bits. Some caveats to keep in mind are that char in C++ is defined as an 8-bit number data type that can be used to store characters, but can also be used to store numbers.

Java is a popular, multiplatform, object-oriented programming language. Java is widely used to code web applications, Android applications, and back-end parts of software. Languages HTML

HTML (Hypertext Markup Language) is a markup language used to create web pages and display information in internet browsers. HTML is an internet standard defined and controlled by the World Wide Web Consortium (W3C).

HTML (Hypertext Markup Language) was developed by :

  • Tim Berners-Lee: The creator of HTML who released the first version in 1993. Berners-Lee is also known as the creator of the world wide web (www).

World Wide Web Consortium (W3C) :
  • Develops versions of HTML after the first version, such as HTML 3.2 which includes new elements for tables, applets, and page layout features.

WHATWG (Web Hypertext Application Technology Working Group) :
  • Developed the HTML5 version.

HTML is a standard markup language used to create web pages and web applications. The most widely used version of HTML in the 2000swas HTML 4.01, which became an official standard in December 1999. HTML consists of tags, elements, and attributes that are interrelated :
  • Tags : Specialized text that tags the content. Tags are created with open square brackets <....> and closed use slashes or /. Tags have different functions, such as creating titles, paragraphs, headings, bold, and italics.
  • Element : The entire HTML code starting from the opening tag to the closing tag.
  • Attributes : Additional information on the element written in the opening tag section.

    • The basic structure of HTML is case insensitive, meaning there is no difference between capital and lowercase letters.

Languages CSS

Cascading Style Sheets (CSS) diciptakan oleh World Wide Web Consortium (W3C) pada tahun 1996. Namun, ide CSS pertama kali diusulkan oleh Håkon Wium Lie pada 10 Oktober 1994. Saat itu, Lie sedang bekerja di CERN bersama Tim Berners-Lee. CSS (Cascading Style Sheets) is a programming language used to determine the appearance and format of websites. CSS serves to describe how elements should be displayed on screen, on paper, in speech, or in other media. CSS can be used to :

  • Set the page layout
  • Set colors
  • Set fonts
  • Set the background
  • Separating content from its visual appearance on the site
  • Make the appearance of web pages neat and optimized on various user screen sizes
  • Improve the accessibility of web content
  • Speed up web page loading

CSS is closely related to HTML, which is a markup language. CSS can complement HTML so that the appearance of a website becomes more attractive. CSS is divided into several types, including: Inline Style CSS, Internal CSS, External CSS. CSS is one of the important skills for Web Developers and Data Scientists.
